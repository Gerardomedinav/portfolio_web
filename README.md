Este repositorio alberga la infraestructura y el código fuente de mi portafolio profesional. El objetivo central de este proyecto no es solo mostrar mi trabajo, sino servir como un **caso de estudio en optimización de rendimiento (Performance), accesibilidad y endurecimiento de seguridad (Hardening)**.

---

## ⚡ Rendimiento y Auditoría (Performance)
El sitio ha sido auditado utilizando **Google PageSpeed Insights (Lighthouse)**, logrando métricas que lo sitúan en el percentil más alto de eficiencia web:

* **Performance: 96%** – Optimización de carga crítica con un LCP (Largest Contentful Paint) de solo **0.7s**.
* **Accessibility: 100%** – Estructura semántica perfecta, garantizando acceso total a tecnologías de asistencia.
* **Best Practices: 92%** – Uso de protocolos modernos y código libre de vulnerabilidades conocidas.
* **SEO: 91%** – Configuración avanzada de meta-tags y arquitectura de datos para indexación orgánica.

---

## 🛡️ Ciberseguridad y Hardening
Se realizó una evaluación de riesgos basada en el marco **NIST SP 800-30**, logrando elevar la calificación de seguridad de **D** a **A** mediante la implementación de cabeceras de seguridad de nivel empresarial:

### Pruebas de Seguridad (Security Headers):
* **Content Security Policy (CSP):** Mitigación proactiva contra ataques de inyección (XSS).
* **X-Frame-Options (DENY):** Protección total contra ataques de Clickjacking.
* **X-Content-Type-Options (nosniff):** Prevención de vulnerabilidades por confusión de tipos MIME.
* **HSTS & TLS 1.3:** Forzado de túneles de comunicación cifrados de alta velocidad.

---

## 🛠️ Tecnologías Aplicadas al Proyecto
Para lograr estos resultados, el portafolio utiliza una combinación estratégica de herramientas de infraestructura y optimización:

* **Infraestructura:** Desplegado en la red de borde (Edge Network) de **Vercel** para minimizar la latencia global.
* **Seguridad de Servidor:** Configuración avanzada mediante `vercel.json` para la inyección de cabeceras HTTP seguras.
* **Optimización Visual:** Implementación de **AOS (Animate On Scroll)** optimizado para no penalizar el tiempo de interactividad (TTI).
* **Análisis de Datos:** Integración de **Google Analytics (GA4)** mediante **Tag Manager** para el monitoreo de métricas de usuario sin afectar el rendimiento del hilo principal del navegador.
